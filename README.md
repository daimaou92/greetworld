# greetworld
A node.js module that just greets the world

## Installation
```sh
npm install greetworld --save
```

## Usage

### Javascript
```javascript
var _g = require("greetworld");
_g.Greet();
```
```sh
Output ---> 'HELLO WORLDDDDDDD!!'
```

### Typescript
```typescript
import {Greet} from 'greetworld';
Greet(); 
```
```sh
Output ---> 'HELLO WORLDDDDDDD!!'
```
